# coffee-shop-website-
Coffee shop web app built with PHP, MySQL, and admin dashboard.

##  Features

###  User Side
- User registration and login
- Browse products by category
- Add products to cart
- Place orders
- View order history
- Contact form to send messages to admin

###  Admin Dashboard
- Admin login with role-based access
- View total users, sold products, and revenue
- Manage products (add / delete)
- View and manage orders
- View contact messages from users
- Secure admin routes

---

##  Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP (Procedural)
- **Database:** MySQL
- **Server:** Apache (XAMPP)
- **Icons:** Font Awesome
